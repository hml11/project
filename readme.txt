邦购网页的制作

使用方式：github 地址https://github.com/hml11/project
使用gulp指令gulp qwe 运行

首页：http://localhost:8383
源文件存在src文件夹，
项目说明：该项目制作的是一个电商网站，分为五个页面：首页，登录注册，商品列表，商品详情，购物车页面。
项目亮点：该项目中的商品列表都是数据驱动页面实现，首页中使用了楼层，轮播图和选项卡，懒加载，二级菜单的效果，商品详情页有放大镜效果
项目运行环境：本地服务器下Node环境
项目的技术栈：该项目使用了jQuery ajax前后端数据交互技术，gulp前端自动化管理工具和git版本管理工具等
